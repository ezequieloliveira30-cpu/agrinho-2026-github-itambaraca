# 🌾 Agro Forte 2026 - Projeto Agrinho

## 📋 Sobre o Projeto

**Agro Forte** é um website educativo desenvolvido para o **Projeto Agrinho 2026 - Nível 3**, com o tema **"Agro Forte: Sustentabilidade na Agricultura"**.

O projeto demonstra como a sustentabilidade agrícola é fundamental para fortalecer o agronegócio, combinando práticas sustentáveis, inovação tecnológica e responsabilidade ambiental.

## 🎯 Objetivo

Criar um website interativo e moderno que:

✅ Apresente o objetivo do projeto de forma clara e inspiradora  
✅ Explique sobre a agricultura em Itambaraca, RS  
✅ Mostre a localização de Itambaraca no mapa  
✅ Ofereça um quiz interativo sobre sustentabilidade (nível fácil)  
✅ Possua modo escuro personalizável  
✅ Tenha opção de aumentar o tamanho das letras  
✅ Apresente dicas práticas de sustentabilidade  
✅ Inclua créditos dos criadores  
✅ Utilize animações e interatividade modernas  

---

## 🎨 Características Principais

### 1. Design Moderno e Responsivo
- Paleta de cores profissional com tons verdes (agricultura) e tons quentes
- Layout responsivo que funciona em desktop, tablet e mobile
- Animações suaves e transições elegantes

### 2. Seções do Website

#### Objetivo
- Apresentação clara do projeto
- Destaque dos três pilares: Sustentabilidade, Força Econômica e Responsabilidade Social
- Design atrativo com gradientes e animações

#### Itambaraca
- Informações sobre a localização de Itambaraca (RS)
- Mapa interativo do Google Maps
- Dados sobre economia, clima e principais culturas
- Estatísticas agrícolas da região

#### Quiz Interativo
- 6 perguntas sobre sustentabilidade agrícola (nível fácil)
- Feedback imediato sobre respostas
- Sistema de pontuação com barra de progresso
- Mensagens personalizadas de resultado
- Opção de refazer o quiz

#### Dicas de Sustentabilidade
- 6 dicas práticas ilustradas com emojis
- Rotação de culturas, Irrigação eficiente, Compostagem
- Tecnologia agrícola, Biodiversidade, Energia renovável

- Nomes do aluno e professor

### 3. Funcionalidades Interativas

## 🚀 Como Usar

### 1. Clonar o Repositório
```bash
git clone https://github.com/ezequieloliveira30-cpu/agrinho-2026-github-itambaraca.git
cd agrinho-2026-github-itambaraca
```

### 2. Abrir o Website
- Abra o arquivo `index.html` diretamente no navegador
- Ou use um servidor local

```bash
python -m http.server 8000
# ou
npx http-server
```

### 3. Acessar no Navegador
- Visite `http://localhost:8000`

## 🎮 Guia de Funcionalidades

### Navegação
- **Logo**: Pula para o início do site
- **Botão 🌙**: Alterna modo escuro/claro
- **Botão A+**: Aumenta tamanho das letras
- **Créditos**: Link para a seção de créditos

### Quiz
1. Clique em "Iniciar Quiz"
2. Escolha uma resposta entre as 4 opções
3. Receba feedback imediato (verde = correto, vermelho = errado)
4. Veja seu score final e mensagem personalizada
5. Clique em "Fazer Quiz Novamente" para tentar novamente

## 🌍 Informações sobre Itambaraca

**Itambaraca** é um município localizado no **norte do Rio Grande do Sul**:

- 🌾 **Principais Culturas**: Soja, milho, trigo e produtos orgânicos
- 🐄 **Pecuária**: Criação de gado de corte e leite
- 🌱 **Características**: Clima subtropical úmido, solo fértil
- 🏡 **Economia**: Agricultura e pecuária
- ♻️ **Sustentabilidade**: Práticas alinhadas com conservação ambiental
